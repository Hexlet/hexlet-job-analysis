# job-analysis

```bash
npm i -g @hexlet-job-analysis
analyze-jobs init
analyze-jobs download <term>
// Example: analyze-jobs download php
// Example: analyze-jobs download java
analyze-jobs normalize
analyze-jobs analyze <term>
// Example: analyze-jobs download java
```
